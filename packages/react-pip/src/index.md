## react-pip

画中画 react 组件

start： 开始动画

stop: 停止动画

toggleDirection：切换动画方向

## 组件方法

| 方法名               | 描述         | 参数                      |
| -------------------- | ------------ | ------------------------- |
| start                | 开始动画     | -                         |
| stop                 | 停止动画     | -                         |
| toggleOrSetDirection | 改变动画方向 | 不填或"forward"或"return" |
